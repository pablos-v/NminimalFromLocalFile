# Приложение для вывода минимального значения N из 1 столбца указанной таблицы Excel 
## Работает в веб-интерфейсе Swagger

Используется первый по счёту лист файла Эксель.

N считается от 1, не от 0.

При валидации не числовые значения типа формул в файле и пустые ячейки игнорируются, строковые ячейки он приводит к числу.

Путь к файлу должен быть абсолютным.

Примеры файлов есть в папке resources.

## Инструкция по сборке и запуску кода
Предусловия для запуска: нужно установить git и maven

Шаги для запуска в консоли:

1. перейти в ваш_путь_к_директории_загрузки

2. скачать приложение командой git clone https://github.com/pablos-v/NminimalFromLocalFile

3. перейти в .\NminimalFromLocalFile\n_minimal

4. запустить команду mvn spring-boot:run

Интерфейс Swagger будет доступен по адресу:
http://localhost:8080/swagger-ui/
